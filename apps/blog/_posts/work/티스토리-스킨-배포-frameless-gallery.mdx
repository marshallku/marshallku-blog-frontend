### 주요 기능

\- Lazy Load : 화면 밖에 있는 이미지의 로딩을 지연해 훨씬 빠른 블로그 제작 가능  
\- 이미지 갤러리 : 본문의 이미지를 한 번에 모아볼 수 있도록 해주는 기능  
\- 이미지 크게 보기 : 본문의 이미지를 전체 화면으로 볼 수 있도록 해주는 기능  
\- 손쉬운 사용자화 : 스킨 설정을 통해 취향에 맞는 블로그를 손쉽게 제작하실 수 있습니다.  
\- 헤더 카테고리 : 페이지 최상단에 최근 글과 썸네일을 카테고리별로 정리해서 표시해주는 기능  
\- jQueryless : jQuery를 사용하지 않아 훨씬 우월한 성능을 보여줍니다.

### 상세 설명

스킨의 가이드는 [이곳](https://frameless-gallery.tistory.com/category/guide)에서 연재할 예정입니다.  
초보자의 눈높이에서 최대한 단순하게 스킨을 활용하는 방법들을 기술하겠습니다.

![](/images/2019/11/fg-image-gallery.jpg)

본문의 이미지를 모아보는 이미지 갤러리 기능입니다.  
스크롤을 많이 하지 않고도 본문에 첨부된 이미지를 편하게 다시 볼 수 있습니다.

개선된 이미지 크게 보기 기능입니다.  
브라우저 창도 표시하지 않고 화면에 꽉 채워 이미지를 표시합니다.

작은 이미지를 강제로 늘리진 않고 원본 크기를 표시하는 것이니, 작은 이미지가 늘어날까 염려하지 않으셔도 됩니다.

![](/images/2019/11/fg-features.png)

추가한 기능 대부분은 코딩 전혀 없이 스킨 편집에서 수정하실 수 있습니다.  
로그인 안 한 사용자의 프로필 이미지 변경, 댓글 최신순 정렬 등 캡처 되지 않은 기능도 많으니 참고해주세요.

![](/images/2019/11/fg-header-category.png)
