## 이미지 뷰어

![](/images/2020/01/image-viewer.jpg)

이미지를 클릭했을 때 새 창에서 열리지 않고 현재 페이지에서 새로운 레이어 위에 이미지가 표시되도록 변경했습니다.

확대 / 축소가 새 창에서 열릴 때 훨씬 편해서 어떤 방식을 써야 할지 고민입니다.

## 비디오 포스트 추가

![](/images/2020/01/video-page.jpg)

[이 포스트](/%f0%9f%92%95/twice/i-feel-so-special)에 임시로 적용해뒀습니다.

문자 그대로 비디오 중심의 포스트를 작성할 때 쓸 수 있는 양식입니다.

\[임시\]인 이유는, 모바일 기기에서 렉 / 화면 부족함이라는 문제를 해결하지 못해서 입니다.

제 휴대폰에선 키보드가 열려도 별문제가 없는데, 디스플레이가 작은 휴대폰에서 키보드가 열렸을 때 화면에 공간이 거의 사라질지도 몰라서 일단 이래저래 여러 기기에서 테스트해보고 사용을 시작할 예정입니다.

## Lazy Load 재추가

클라이언트 사이드에서 lazy laod를 사용하도록 업데이트했습니다.  
RSS 피드, 검색 엔진에 아무런 영향을 끼치지 않고 lazy load가 작동합니다.

검색 엔진에 영향을 최소화하기 위해 봇이 크롤링할 땐 작동을 막아두기까지 했습니다.

## 댓글 관련

댓글 작성 후, 아무런 오류 없이 댓글이 잘 작성되면 본인이 작성한 댓글로 스크롤이 이동합니다.

지금까지 오류가 발생한 적이 없었지만, 행여 오류가 발생하면 어떤 오류가 발생했는지 유저에게 알림이 가도록 업데이트했습니다.  
댓글 작성 중 서버가 죽거나 인터넷 연결이 끊기는 게 아니면 보실 일은 없지 싶습니다.

댓글의 이메일도 a 태그로 감싸도록 업데이트했습니다.
