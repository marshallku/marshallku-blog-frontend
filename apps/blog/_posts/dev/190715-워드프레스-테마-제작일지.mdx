홈 화면을 조금 더 꾸며봤습니다.  
예전 티스토리처럼 특정 카테고리의 글을 표시하는데, 아직 쓴 포스트가 별로 없어 뭘 좀 많이 추가하진 못했습니다.

작업하며 썸네일이 배경으로 들어가는 리스트도 만들었습니다. 예전엔 커버에만 사용할 수 있었지만, 이번엔 카테고리 글 목록 등에도 사용할 수 있도록 했습니다.

![](/images/2019/07/1-18.png)

헤더에 카테고리 글 목록을 표시하고, 해당 카테고리에 3일 내로 쓴 글이 있으면 동그라미 하나가 표시되도록 했습니다.

![](/images/2019/07/2-14.png)

좌측 드로워에 최근 댓글을 추가했습니다.

![](/images/2019/07/3-11.png)

Masonry 리스트의 hover 효과를 변경했습니다.

이미지가 blur 처리되고 투명해지며 글의 정보가 드러나는 방법에서, 이미지 중앙에서 검은색 원이 퍼지고 글의 정보가 드러나는 방식으로 변경했습니다.

![](/images/2019/07/4-8.png)

포스트 화면 제일 아래에 해당 카테고리의 현재 읽고 있는 글을 제외한 최신 글이 6개씩 표시됩니다.
