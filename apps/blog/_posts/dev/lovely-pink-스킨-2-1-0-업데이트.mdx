![홈 화면 2](/images/2020/12/home2.png)

먼저, 글꼴을 인피니티산스와 가나초코로 변경했습니다. 기본적으로 인피니티산스를 사용하고, 제목에 가나초코를 사용합니다.

또한, 썸네일이 조금 덜 각지게 표시되게 업데이트했고, 슬라이더의 디자인을 조금 다듬었습니다.

![드로어](/images/2020/12/drawer.png)

드로어를 여는 버튼에 커서를 올리면 화살표로 변하는 애니메이션을 추가하고, 카테고리 상단에 구분 선을 추가했습니다. 하위 카테고리 목록에도 배경을 추가해 구분이 뚜렷하여지게 했습니다.

![방명록](/images/2020/12/guestbook.png)

댓글 / 방명록을 입력하는 부분의 여백을 통일감 있게 수정했습니다. 또한, 비밀 댓글 버튼과 댓글 입력 버튼의 글자를 삭제하고 아이콘만 표시되도록 수정했습니다.  
댓글에 링크를 추가하는 기능을 0.5초마다 실행하지 않고 답글 추가 / 댓글 수정 등의 이벤트가 발생했을 때 실행하도록 업데이트했습니다.

또한, 방명록의 프로필 사진을 넣는 방식을 자바스크립트가 아닌 치환자로 변경했습니다.

외에도 글자 크기 조정, css / js 파일 불러오는 방식 최적화, js 난독화 해제 등의 업데이트가 진행되었습니다.
