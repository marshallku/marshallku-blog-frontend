꽤 예전에 업데이트하고 방치 중이었는데, 유저 수가 제가 마지막으로 확인했을 때보다 3배 가까이 늘었네요.  
어떻게 늘었는진 저도 잘 모르겠습니다…

---

여하튼, 새 앨범도 나왔으니 사진이라도 업데이트할까 싶어 오랜만에 vscode에서 열었다가, 서버도 있는데 [트와이스 팬 페이지](https://marshallku.com/twice/)나 [Canvas 썸네일 제작기](https://marshallku.com/canvas-thumbnail-maker)처럼 서버에다 페이지를 넣어도 괜찮지 않을까 싶어서 반응형으로 디자인을 업데이트하는 중입니다.

제 서버로 페이지 옮겼을 때 장점  
\- 모든 기기, 모든 브라우저에서 접근할 수 있어진다.

제 서버로 페이지 옮겼을 때 단점  
\- 크롬 서버 사용이 불가능해진다.

단점은 그냥 로컬 스토리지에 정보를 저장하면 끝날 문제고, 확장 프로그램이랑 페이지 둘 다 업데이트하면 그냥 없는 단점이나 마찬가지 같네요.

![](/images/2019/09/twice-new-tab-mobile.png)

PC에서만 사용 가능해서, 화면 크기가 작은 기기에서 사용하는 건 생각도 않았던 터라, 화면 크기가 작아지면 레이아웃이 박살이 납니다.

![](/images/2019/09/twice-new-tab-mobile-friendly.png)

폰트랑 특정 요소 너비만 줄여줘도 크게 이상하진 않더라고요.

![](/images/2019/09/twice-new-tab-mobile-setting-1.png)

세팅도 max-width 적절히 설정해주면 틀이 망가지진 않는데, 체크박스가 이상하게 표시되더라고요…😥  
아마 이건 원활한 표시를 위해 모바일에선 표시하는 방법을 아예 뜯어고치던지 해야겠습니다.
