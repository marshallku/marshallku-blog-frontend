사용하는 서버가 75개가 넘어갔을 때, 100개를 넘기면 인증 마크 달아준다고 인증하라고 문자까지 보내서 당시에 ID 카드(신분증)만 인증했었습니다. 시간이 아무리 지나도 인증이 안 되길래 뭔가 잘못됐다 싶어 들어가 보니 신분증에 이것저것 답변까지 추가로 해줘야 하는 거였더라고요.  
봇의 목적 같은 간단하고 당연한 것부터, 어떤 유저 정보를 저장하는지, 저장 기간은 어떻게 되는지, 유저가 정보를 지우려면 어디로 연락해야 하는지, 어떤 서버를 쓰는지, 어떤 방식으로 보안 체계를 구성했는지 등등 다양한 것들을 물어보더라고요.

OAuth를 사용하나, 만약 사용하면 어떻게 쓰냐 뭐 그런 질문에 No.라 대답하니 100자 안 넘긴다고 채워 오라던 깐깐함에 안되면 어떡하나 걱정하며 일주일을 보낸 결과

![디스코드 봇 검증](/images/2020/11/discord-bot-verification.png)

오늘 이런 메일을 받았습니다.

![아이유 봇 인증](/images/2020/11/Verified-IU-Bot.png)

인증된 봇 딱지가 붙고, 101개 이상의 서버에서도 사용할 수 있게 되었습니다.

Node.js 연습용으로 만든 봇인데, 일이 이렇게 될 줄은 몰랐네요.  
클라우드 컴퓨터 사용기간 1년 끝나면 문 닫을 생각이었는데, 그러면 안 될 것 같은 분위기네요. 저희 서버에서만 쓸 줄 알고 만든 봇이었는데 이렇 일이 이렇게 커질 줄은…

아무튼 감사합니다. 😊
