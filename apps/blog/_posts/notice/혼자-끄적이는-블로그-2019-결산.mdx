### 3번의 레이아웃 업데이트가 있었습니다.

![](/images/2019/12/블로그-레이아웃-2019-01.png)

2019년 1월 11일 기존의 Summer Nights를 전부 갈아엎고 2.0버전으로 업데이트를 시작했습니다.

![](/images/2019/12/블로그-레이아웃-2019-02.png)

다시 기존의 스킨을 모두 갈아엎고 새로 업데이트를 진행했습니다.  
Summer Nights 3.0은 너무 성의 없어 보여서 아직도 아무런 이름이 없는 스킨입니다.

![](/images/2019/06/0.png)

티스토리에서 홈 서버로 이사를 진행하며 티스토리 스킨에서 워드프레스 테마로 변경을 진행했습니다.  
외적인 변화는 적었어도 내적인 변화는 아주 많았던 업데이트입니다.

여담으로 메인 css 파일은 월평균 100회, 메인 js 파일은 월평균 60회 업데이트를 진행해왔습니다.

### 홈 서버 가동을 시작했습니다.

![](/images/2019/12/DSCF0187-downscaled.jpg)

서버사이드 언어의 공부를 시작하며 진정한 나만의 웹사이트 제작을 위해 6월부터 홈 서버를 가동했습니다.

중간에 CPU가 맛이 가서 보드, CPU, RAM 전부 갈아야 하는 사태도 있었고, 갖은 오류로 저를 힘들게 하지만 반년 만에 애증의 관계가 돼버렸네요.

### jQuery를 블로그에서 제거했습니다.

2016년부터 시간이 멈춰버린 IE를 아예 신경 끄기로 마음먹고 블로그에서 jQuery를 걷어냈습니다.

jQuery를 걷어내며 블로그의 스크립트도 대규모로 업데이트를 진행해, 예전에 운영하던 블로그의 스킨보다 훨씬 적은 에러, 훨씬 빠른 속도로 스크립트가 실행되도록 했습니다.
